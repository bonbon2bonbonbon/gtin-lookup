GTIN LOOKUP
브랜드별 사이즈별 GTIN(EAN) 조회 — 쿠팡 등록용
STEP 1
품번 입력 후 API 열기
클릭 시 새 탭에서 아디다스 API가 열려요
STEP 2
새 탭 전체 복사 후 붙여넣기
Ctrl+A 전체선택 → Ctrl+C 복사 → 여기서 Ctrl+V
STEP 1
스타일코드 입력 후 상품 페이지 열기
클릭 시 새 탭에서 나이키 상품 페이지가 열려요
STEP 2
페이지 소스 열고 전체 복사 후 붙여넣기
새 탭에서 Ctrl+U (소스보기) → Ctrl+A → Ctrl+C → 여기서 Ctrl+V
※ 상품 페이지가 아니라 소스 페이지(Ctrl+U)에서 복사해야 해요!
안내
뉴발란스 — 쿠팡 RA번호 발급
뉴발란스 공홈은 GTIN을 외부에 노출하지 않아요.
쿠팡에서 RA번호를 발급받아 대신 사용하세요.

쿠팡 Wing → 상품등록 → 바코드 없음 선택 → RA번호 자동 발급

→ 뉴발란스 공홈 바로가기

쿠팡에서 RA번호를 발급받아 대신 사용하세요.

쿠팡 Wing → 상품등록 → 바코드 없음 선택 → RA번호 자동 발급

→ 풋조이 공홈 바로가기
